## 데이터 저장

### 로컬 저장
모든 스캔 문서, 이미지, 설정은 다음 위치에 저장됩니다:
- 앱 샌드박스 (iOS 보안 영역)
- 사용자가 명시적으로 선택한 경우 iCloud (Apple이 관리)

### 데이터 보안
- iOS 시스템 보안 기능에 의해 보호
- 다른 앱의 접근 불가
- 사용자만 접근 및 삭제 가능

## 데이터 공유

MeloCatScan은 제3자와 어떠한 데이터도 공유하지 않습니다.

### 사용자 주도 공유
사용자가 명시적으로 선택한 경우에만:
- PDF/이미지 내보내기
- 텍스트 클립보드 복사
- iOS 공유 시트를 통한 공유

## 제3자 서비스

MeloCatScan은 다음을 사용하지 않습니다:
- 광고 네트워크
- 분석 도구
- 소셜 미디어 SDK
- 크래시 리포팅 서비스 (App Store Connect 기본 제공 제외)
